# Планирование тестирования
  ### Функциональное тестирование
- Проверить корректность работы всех типов вопросов в квизах
- Убедиться в правильности подсчета баллов
- Проверить работу корзины товаров
- Тестирование адаптивности интерфейса

  ### Пользовательское тестирование (UX)
- Оценить удобство навигации
 -Проверить интуитивность интерфейса
- Выявить сложности в использовании
- Собрать предложения по улучшению

  ### Тестирование производительности
- Проверить время загрузки страниц
- Тестирование отзывчивости интерфейса
- Проверить работу на медленных соединениях

## Область тестирования

### Включено в тестирование:
- Компоненты создания квизов
- Компоненты прохождения квизов
- Система корзины товаров
- Навигация между страницами
